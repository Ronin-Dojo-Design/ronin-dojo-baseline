# Technique - Jab

## Teaching cues
- Extend from the floor up.
- Keep the shoulder relaxed.
- Return the hand on the same path.

## Prerequisites
- stable stance
- guard position

## Counters and risks
- can be slipped if telegraphed
- can expose centerline if overextended
