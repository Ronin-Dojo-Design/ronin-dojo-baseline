## What we're building

- A markdown-outline to branded HTML slide-deck renderer
- Zero external assets — one self-contained HTML file
- 16:9 slides with keyboard and click navigation

Notes: Emphasize that this is standalone tooling, not a dependency on render-doc.

## Why it matters

> Good tools compound. A deck you can write in a text editor and ship in one file beats a slide app you have to fight.
