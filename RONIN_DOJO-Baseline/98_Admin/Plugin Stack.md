# Plugin Stack

## Core first
- Daily Notes
- Templates
- Properties
- File Explorer
- Obsidian URI
- Web Clipper

## Then add structure
- Tasks for task aggregation and editing
- Dataview for live tables and dashboards
- Templater for dynamic note creation

## Optional later
- Bases for database-like views
- Canvas for mapping modules, flows, and relationships visually

## Rule of thumb
Start with the smallest stack that changes your behavior.
Do not install ten plugins before you can create, link, and review five clean notes.
