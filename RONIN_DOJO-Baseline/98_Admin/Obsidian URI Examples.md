# Obsidian URI Examples

Replace `Ronin Vault` with your actual vault name.

## Open the command center
`obsidian://open?vault=Ronin%20Vault&file=02_Dashboards%2FCommand%20Center`

## Create a new inbox note silently
`obsidian://new?vault=Ronin%20Vault&file=00_Inbox%2FQuick%20Capture&content=%23%20Quick%20Capture&silent=true`

## Append a task to the backlog
`obsidian://new?vault=Ronin%20Vault&file=09_Tasks%2FProject%20Backlog&append=true&content=-%20%5B%20%5D%20Review%20new%20curriculum%20idea`

## Open today's daily note
`obsidian://daily?vault=Ronin%20Vault`
