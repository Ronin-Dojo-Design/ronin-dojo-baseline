# Ronin Obsidian Content Engine Pack

This pack turns Obsidian into a **content atom + production workflow system** that can feed Pods, React components, curriculum pages, and social variants.

Included:
- `02_Dashboards/Content Engine Center.md`
- `12_Content_Engine/Content Atom Template.md`
- `12_Content_Engine/Distribution Variant Template.md`
- `12_Content_Engine/Publication Runbook Template.md`
- `12_Content_Engine/Component Registry Template.md`
- `98_Admin/Ronin Shared Content Architecture.md`
- `99_Code/JETTY React + API Examples.md`

Recommended use:
1. Create one **Content Atom** per idea, lesson, seminar, interview, or technique topic.
2. Store the teaching truth once.
3. Derive social, blog, curriculum, and video outputs from the same note.
4. Publish selected fields into Pods.
5. Keep task state and publication history in the Ronin operational layer.
