Do / Don't

✅ Do

  • Use brand red #E52421 as the single accent.
  • White number on a red disc for steps.
  • Headings Poppins extrabold uppercase italic.
  • Import scripts/lib/bbl-doc-theme.ts in generators.

❌ Don't

  • Use gold #d7a74c / #FFD700 — styles.css:210 records this exact import as a defect already corrected once.
  • Hand-roll a second token block.
  • Outline-style step discs.
  • Exception: the Galaxy viewer's star palette is intentionally gold — art inside the 3D scene, not brand chrome.
