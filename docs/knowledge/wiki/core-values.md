# Core Values Ledger (CV)

ADR-adjacent value rows (`CV-NNN`) governing how docs, wiki, runbooks, SOPs, protocols, and
rituals are written platform-wide. LLL-shaped: rows, not essays. Apply to **new and touched**
docs only — no retroactive sweep. Ratification bar: survive an operator grill + three practical
uses (see the vault `MMB_GRILL` MMB-Q-006 trail).

| ID | Value | Definition (1 line) | Applies to | Ratified |
| --- | --- | --- | --- | --- |
| CV-001 | **EEE** — Effective · Efficient · Excellent | Achieve the intended result, remove avoidable work, keep evidence/quality high — brand-aware, prose-minimal. | docs · wiki · runbooks · SOPs · protocols · rituals | SESSION_0572 grill (MMB-Q-006 A) |
| CV-002 | **TD** — Token Discipline | Spend context on decisions and proof: links, IDs, tables, compact pickup state — never trade away safety, authority, or required verification. | all agent-read docs + session output | SESSION_0572 grill (MMB-Q-006 A) |
